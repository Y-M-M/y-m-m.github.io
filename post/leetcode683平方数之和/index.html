leetcode 683 平方数之和

题目

给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c 。

示例

示例一

输入：c = 5 输出：true 解释：1 * 1 + 2 * 2 = 5

示例二

输入：c = 3 输出：false

提示

0 <= c <= 231 - 1

题解

方法一

class Solution {
public:
    bool judgeSquareSum(int c) {
        unordered_set<int> little_squares;
        long long base = 0;
        long long square = base * base;
        while (square <= c / 2) {
            little_squares.insert(square);
            if (square + square == c) return true;
            base++;
            square = base * base;
        }
        while (square <= c) {
            if (little_squares.count(c - square)) return true;
            base++;
            square = base * base;
        }
        return false;
    }
};

  • 提交结果

    • 通过

  • 分析

    • 思路
      • 计算不超过c/2的所有平方数，存入unordered_set中，便于查找
      • 计算超过c/2且不超过c的所有平方数x，在unordered_set中查找是否有c - x，如果有，返回true
      • 单独处理c/2刚好是平方数的情况
    • 时间复杂度 O(sqrt(n))
      • unordered_map()节省了查找的时间
    • 空间复杂度 O(sqrt(n))

方法二

class Solution {
public:
    bool judgeSquareSum(int c) {
        for (long a = 0; a * a <= c; a++) {
            double b = sqrt(c - a * a);
            if (b == (int)b) {
                return true;
            }
        }
        return false;
    }
};
  • 提交结果
    • 通过
  • 分析
    • 使用sqrt函数，查看是否有整数的平方根
    • 时间复杂度 O(sqrt(n))
    • 空间复杂度O(1)

方法三

class Solution {
public:
    bool judgeSquareSum(int c) {
        long left = 0;
        long right = (int)sqrt(c);
        while (left <= right) {
            long sum = left * left + right * right;
            if (sum == c)
                return true;
            else if (sum > c)
                right--;
            else
                left++;
        }
        return false;
    }
};
  • 提交结果
    • 通过
  • 分析
    • 双指针
    • 时间复杂度O(sqrt(n))
    • 空间复杂度O(1)

方法四

class Solution {
public:
    bool judgeSquareSum(int c) {
        for (int base = 2; base * base <= c; base++) {
            if (c % base != 0) continue;
            int exp = 0;
            while (c % base == 0) // 计算base的幂
            {
                c /= base;
                exp++;
            }
            if (base % 4 == 3 && exp % 2 != 0) {
                return false;
            }
        }
        return c % 4 != 3;
    }
};
  • 提交结果
    • 通过
  • 分析
    • 数学
    • 费马平方和定理
    • 一个非负整数 c 如果能够表示为两个整数的平方和，当且仅当 c 的所有形如 4k+3 的质因子的幂均为偶数。
    • 依次遍历所有质因子
      • 每次都让c /= base，则遍历到的因子一定是质因子(如果因子除了1和它本身之外还有其他因子，则一定会在前面被除掉)
    • 对每个质因子计算幂，判读它是否符合条件
    • 时间复杂度 O(sqrt(n))
    • 空间复杂度 O(1)
