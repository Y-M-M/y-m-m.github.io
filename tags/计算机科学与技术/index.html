  • 并查集

    什么是并查集？

    • 并查集是一种用于管理元素所属集合的数据结构，实现为一个森林，其中每棵树表示一个集合，树中的节点表示对应集合中的元素。

    核心思想

    • 提取树的重点特征：父亲结点->祖先结点
    • 拥有相同的祖先节点的子节点属于同一个集合
    • 被边连通的两个节点并入同一个集合(两棵树的合并)
    • 为了简化操作，可将将节点与其祖先节点直接连通，将祖先节点作为其父亲节点，即压缩操作

    并查集的创建

    • 创建father数组，存储每个节点的父亲节点，初始的父亲节点为它本身
    • 对于每一条边，取它连通的两个节点；对每一个节点，通过Find函数返回其祖先节点，并进行路径压缩

    Find()

    int Find(vector<int>& parent, int index) { // 返回祖先节点
        if (parent[index] != index) {
            parent[index] = Find(parent, parent[index]); // 路径压缩
        }
        return parent[index];
    }
    • 当边连通的两个节点祖先不同时，进行连通操作
    void Union(vector<int>& parent, int index1, int index2) { // 如果边连接的两个节点祖先不同，则将其中一个节点的祖先并入另一个节点
        parent[Find(parent, index1)] = Find(parent, index2);
    }

    元素的删除

    • 将要删除的结点的父亲节点设为它自己即可

  • python dictionary的使用

    什么是dictionary?

    • 字典是Python中的一种映射类型
    • 每一个元素都是一个键值对
    • 字典中的元素通过键进程索引
      • 可以通过键直接访问对应的值
      • 字典中的键是唯一的
      • 字典中的元素(键值对)是无序的

    字典的内部原理是什么？

    • 哈希映射
    • 故字典键必须是可哈希对象
      • 特别的，如果元组中包含可变数据类型的元素，则该元组不能作为字典中的键

    字典的创建

    键值对赋值

    score = {'计算机组成原理': 86, '计算机网络': 90}

    字典推导式

    {x: x**2 for x in range(10)}

    类型构造函数dict()

    dict() # 创立空字典

dict([(k1, v1), (k2, v2), ...]) # 使用可迭代对象

dict(k1 = v1, k2 = v2, ...) # 使用键值对序列

dict(d1) # 使用其他映射类型对象

    fromkeys方法创建初始值相同的字典

    dict.formkeys(可迭代对象, value)
    • value 默认为none

    字典的基本操作

    读取键值

    d[key] # 获取字典中与key相关联的值，如果key不存在，引发KeyError

    什么是unorderedmap？

    • 一个关联容器，包含具有唯一键的键值对

    使用unorderdmap有什么好处？

    • 元素的搜索、插入和删除时间复杂度O(1)

    原理是什么？

    • 在内部，元素不按任何特定顺序排序，而是组织到存储桶中
    • 元素放入哪个存储桶完全取决于其键的哈希值，具有相同哈希码的键出现在同一个存储桶中

    如何使用？

    初始化

        {"GREEN", "#00FF00"},

  • leetcode 3164 优质对的总数二

    题目

    leetcode 3164 优质对的总数二

    • 给你两个整数数组 nums1 和 nums2，长度分别为 n 和 m。同时给你一个正整数 k。

    • 如果 nums1[i] 可以被 nums2[j] * k 整除，则称数对 (i, j) 为 优质数对（0 <= i <= n - 1, 0 <= j <= m - 1）。

    • 返回 优质数对 的总数。

    示例

    示例一

    输入：nums1 = [1,3,4], nums2 = [1,3,4], k = 1 输出：5 解释： 5个优质数对分别是 (0, 0), (1, 0), (1, 1), (2, 0), 和 (2, 2)。

    题解

    方法一

    class Solution {
public:
    long long numberOfPairs(vector<int>& nums1, vector<int>& nums2, int k) {
        int ans = 0;
        for (long long i = 0; i < nums2.size(); i++) {
            nums2[i] *= k;
        }
        for (long long i = 0; i < nums1.size(); i++) {
            for (long long j = 0; j < nums2.size(); j++) {
                if (nums1[i] % nums2[j] == 0) {
                    ans++;
                }
            }
        }
        return ans;
    }
};

    方法二(力扣官方题解)

    class Solution {
public:
    long long numberOfPairs(vector<int>& nums1, vector<int>& nums2, int k) {
        unordered_map<int, int> store_nums1;
        unordered_map<int, int> store_nums2;
        int max1 = 0; // nums1中的最大的数
        for (int num1: nums1) {
            store_nums1[num1]++;
            max1 = max(max1, num1);
        }
        for (int num2: nums2) {
            store_nums2[num2]++;
        }
        long long ans = 0;
        for (const auto& pair: store_nums2) {
            int a = pair.first, cnt = pair.second;  // 遍历num2
            for (int b = a * k; b <= max1; b += a * k)
            {
                if (store_nums1.count(b) > 0) // 在num1中能找到num2[i]*k的倍数
                {
                    ans += 1L * store_nums1[b] * cnt;
                }
            }
        }

        return ans;
    }
};

      • 使用unordered_map
      • 对于store_nums2中的每个数字乘以k，利用unordered_map直接在nums1中寻找其倍数，而不是遍历store_nums1与store_nums2

  • leetcode 3171 找到按位或最接近k的子数组

    题目

    • 给你一个数组 nums 和一个整数 k 。
    • 你需要找到 nums 的一个子数组，满足子数组中所有元素按位或运算 OR 的值与 k 的 绝对差 尽可能 小 。
    • 换言之，你需要选择一个子数组 nums[l..r] 满足 |k - (nums[l] OR nums[l + 1] … OR nums[r])| 最小。
    • 请你返回 最小 的绝对差值。
    • 子数组 是数组中连续的 非空 元素序列。

    示例

    示例1

    • 输入：nums = [1,2,4,5], k = 3
    • 输出：0
    • 解释：
      • 子数组 nums[0..1] 的按位 OR 运算值为 3 ，得到最小差值 |3 - 3| = 0 。

    示例2

    • 输入：nums = [1,3,1,3], k = 2
    • 输出：1
    • 解释：
      • 子数组 nums[1..1] 的按位 OR 运算值为 3 ，得到最小差值 |3 - 2| = 1 。

    示例3

    • 输入：nums = [1], k = 10
    • 输出：9
    • 解释：
      • 只有一个子数组，按位 OR 运算值为 1 ，得到最小差值 |10 - 1| = 9 。

    提示

    • 1 <= nums.length <= 105
    • 1 <= nums[i] <= 109
    • 1 <= k <= 109

    相关标签

    • 线段树
    • 数组
    • 二分查找

    • 困难

    题解

    解法一

    class Solution {
public:
    int minimumDifference(vector<int>& nums, int k) {
        int n = nums.size();
        vector<int> differ(n);
        for (int i = 0; i < n; i++) {
            differ[i] = abs(nums[i] - k);
        }
        vector<int> or_results(n);
        or_results[0] = nums[0];
        for (int i = 1; i < n; i++) {
            or_results[i] = or_results[i - 1] | nums[i]; 
            int now_differ = abs(or_results[i] - k);
            if (now_differ < differ[i]) {
                differ[i] = now_differ;
            }
            else {
                or_results[i] = nums[i];
            }
        }
        int min_differ = differ[0];
        for (int i = 1; i < n; i++) {
            min_differ = min(min_differ, differ[i]);
        }
        return min_differ;
    }
};
