导论 马克思主义中国化时代化的历史进程与理论成果 Link to heading

马克思主义中国化时代化的提出 Link to heading

马克思是什么时候进入中国的 Link to heading

  • 1899年 清末 传入中国
  • 1917年 十月革命爆发
  • 1919年 五四运动
  • 1921年 中国共产党诞生，中国人民从精神上由被动转为主动，中国选择马克思主义，是历史的选择，是人民的选择
  • 1938年 提出马克思主义中国化，毛泽东在党的六届六中全会上作了《论新阶段》的报告
  • 1945年，在党的七大上，刘少奇代表党中央作了《关于修改党章的报告》，对马克思主义中国化从理论上作了进一步的阐述，并指出毛泽东思想是“中国化的马克思主义”。
  • 2021年，党的十九届六中全会通过的《中共中央关于党的百年奋斗重大成就和历史经验的决议》，总结了百年来中国共产党推进马克思主义中国化时代化的重大成就，阐释了马克思主义中国化时代化的重大历史意义
  • 2022年，当的二十大明确把“不断谱写马克思主义中国化时代化新篇章”作为当代中国共产党人的庄严历史责任，并提出了继续推进马克思主义中国化时代化的新要求

中国选择了马克思主义 Link to heading

  • 是众多先进分子的共同选择
  • 国内探索：毛泽东也是在不断学习和革命实践的过程中，认识到只有马克思主义、只有社会主义才能救中国的深刻道理，从而实现了向马克思主义者的转变
  • 日本：陈独秀、李大钊、胡汉民
  • 美国和苏联

中国共产党成立的意义 Link to heading

  • 改变了近代以后中华民族发展的方向和进程
  • 深刻改变了中国人民和中华民族的前途和命运
  • 深刻改变了世界发展的趋势和格局

马克思主义必须中国化时代化 Link to heading

  • （中国共产党）

马克思主义中国化时代化的内涵 Link to heading

  • 立足中国国情和时代特点，坚持把马克思主义基本原理同中国具体实际相结合，同中国优秀传统文化相结合

运用马克思主义解决中国问题 Link to heading

  • 运用马克思主义的立场、观点和方法，观察时代、把握时代、引领时代，解决中国革命

总结中国经验，发展马克思主义 Link to heading

  • 总结和提炼中国革命、建设、改革的实际经验并将其上升为理论，不断丰富和发展马克思主义的理论宝库，赋予马克思主义新的时代内涵

运用民族语言，阐述马克思主义 Link to heading

  • 运用中国人民喜闻乐见的民族语言来阐述马克思主义，使其根植于中华优秀传统文化的土壤之中，具有中国特色、中国风格、中国气派

马克思主义中国化时代化的历史进程 Link to heading

  • 1921-1949年 新民主主义革命时期 创造了新民主主义革命的伟大成就，为实现中华民族伟大复兴创造根本社会条件
  • 1949-1978年 社会主义革命和建设时期 创造了社会主义革命和建设的伟大成就，为实现中华民族伟大复兴提供充满新的活力的体制保证和快速发展的物质条件
  • 1978-2012年 改革开放和社会主义现代化建设新时期 创造了改革开放和社会主义现代化建设的伟大成就，为实现中华民族伟大复兴提供充满新的活力的体制保证和快速发展的物质条件
  • 2012年至今，中国特色社会主义新时代 创造了新时代中国特色社会主义的伟大成就，为实现中华民族伟大复兴提供了更为完善的制度保证，更为坚实的物质基础，更为主动的精神力量

青年毛泽东（1893年）时期的世界和中国 Link to heading

  • 国内背景

    • 甲午一战：中国知识分子开始普遍觉醒
    • 清末新政，科举废除，新式教育体制开始建立
    • 辛亥革命：鞑虏易驱，共和难建

  • 国际背景

    • 第一次世界大战（1914年），老牌的资本主义国家开始没落，美日崛起

  • 思想背景

    • 唯心->唯物
    • 循环、退化->进化
    • 社会主义
      • 农业社会主义：洪秀全“天朝田亩制度”
      • 大同社会主义：康有为“天下为公”
      • 无政府社会主义：吴稚晖
      • 民生社会主义：孙中山
      • 基尔特社会主义：罗素、张东荪
      • 新村主义or工读互助主义：周作人、王光祈
      • 科学社会主义
    • 其他思潮
      • 体育、武术救国：中华武士会 李存义、张之江
      • 佛教救国：李叔同

毛泽东早年的经历及其思想变化 Link to heading

出乡关 Link to heading
  • 1893年12月26日出生于湖南省湘潭县韶山冲
  • 9岁以前主要在外祖父家度过
  • 14-15从报个文运昌处读了郑观应的《盛世危言》
  • 16岁 维新派教师李漱清
  • 17岁进入东山高等小学堂
  • 孩儿立志出乡关，学不成名誓不还。埋骨何须桑梓地，人生无处不青山。
  • 《新民丛报》，赞成君主立宪，赞成维新派改革，崇拜康有为和梁启超。
师范生 Link to heading
  • 18岁 辛亥革命：投笔从戎
青年时期 Link to heading
  • 19岁 清帝退位，继续求学，《商鞅徙木立信论》（毛泽东最早的手稿），退学自修（《天演论》《民约论》）
  • 《毛泽东早期文稿》
  • 1913年20岁，考入湖南省立第四师范
  • 1917年24岁 《体育之研究》 “文明其精神，野蛮其体魄” 发表于《新青年》
  • 1918年创立新民学会
  • 25岁大学毕业
五四时期 Link to heading
  • 1918年8月-1919年3月 首次到北京协商赴法勤工俭学。北京大学图书管理员
  • 1919年5月，北京学生国内联合会派邓中夏到湖南联络，毛泽东介绍邓中夏雨各校代表见面，成立新的湖南学生联合会
  • 1919年创办《湘江评论》
  • 受到胡适思想的影响，发表《问题研究会章程》列举了几十个问题
  • 受工读互助思想影响，设想在岳麓山设立工读同志会，从事半耕半读
  • 长沙城赵五贞事件，关注女性婚恋自由（新中国成立之后，颁布的第一部法律为婚姻法）
  • 1921年 俄式，此方法较之别的改造房师傅啊所含可能的性质的更多
马克思主义时期 Link to heading
  • 作为中国一大的代表
  • 1921年 湖南自修大学，组织发动工人运动：长沙铅印活板工会、长沙缝纫工会、泥木工会，人力车、理发工人工会
  • 1924年 国民党一大，胡汉民秘书、中央组织部部长、宣传部部长

毛泽东思想形成发展的过程 Link to heading

  • 发端 第一次国内革命战争时期 《中国社会各阶段的分析》《湖南农民运动考察报告》 提出了新民主主义革命的基本思想
  • 形成 土地革命战争时期 《中国的红色政权为什么能够存在？》《井冈山的斗争》《星星之火、可以燎原》 农村包围城市、武装夺取政权道路的提出，标志着毛泽东思想的初步形成
  • 成熟 遵义会议后到战争胜利时期 《实践论》《矛盾论》《新民主主义论》 系统阐述新民主主义革命理论，标志着毛泽东思想趋于成熟
  • 发展 解放战争时期和新中国成立后 《论人民民主专政》《论十大关系》《关于正确处理人民内部矛盾的问题》 关于社会主义革命和社会主义建设的重要思想是毛泽东思想的丰富和发展
