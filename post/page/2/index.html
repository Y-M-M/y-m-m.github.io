    leetcode 1338 数组大小减半

    题目

    给你一个整数数组 arr。你可以从中选出一个整数集合，并删除这些整数在数组中的每次出现。
    返回 至少 能删除数组中的一半整数的整数集合的最小大小。

    示例

    示例一

    输入：arr = [3,3,3,3,5,5,5,2,2,7]
    输出：2
    解释：选择 {3,7} 使得结果数组为 [5,5,5,2,2]、长度为 5（原数组长度的一半）。
    大小为 2 的可行集合有 {3,5},{3,2},{5,2}。
    选择 {2,7} 是不可行的，它的结果数组为 [3,3,3,3,5,5,5]，新数组长度大于原数组的二分之一。

    示例二

    输入：arr = [7,7,7,7,7,7]
    输出：1
    解释：我们只能选择集合 {7}，结果数组为空

    提示

    • 1 <= arr.length <= 105
    • arr.length 为偶数
    • 1 <= arr[i] <= 105

    题解

    方法一

    class Solution {
public:
    int minSetSize(vector<int>& arr) {
        if (arr.size() == 1) return 1;
        vector<int> cnt(100001, 0);
        int r = -1, l = 100001;
        for (int i = 0; i < arr.size(); i++)
        {
            cnt[arr[i]]++;
            if (arr[i] < l) l = arr[i];
            if (arr[i] > r) r = arr[i];
        }
        vector<int> sortedcnt;
        for (int i = l; i <= r; i++)
        {
            sortedcnt.push_back(cnt[i]);
        }
        make_heap(sortedcnt.begin(), sortedcnt.end());
        int ans = 0;
        int num = 0;
        while (num < arr.size() / 2)
        {
            pop_heap(sortedcnt.begin(), sortedcnt.end());
            num += sortedcnt.back();
            ans++;
            sortedcnt.pop_back();
        }
        return ans;
    }
};

    class Solution {
public:
    vector<int> getFinalState(vector<int>& nums, int k, int multiplier) {
        for (int i = 0; i < k; i++)
        {
            int pos = 0;
            int minnum = nums[0];
            for (int j = 1; j < nums.size(); j++)
            {
                if (nums[j] < minnum)
                {
                    pos = j;
                    minnum = nums[j];
                }
            }
            nums[pos] *= multiplier;
        }
        return nums;
    }
};

    方法二

    class Solution {
public:
    vector<int> getFinalState(vector<int>& nums, int k, int multiplier) {
        auto cmp = [&nums](int left, int right) {return nums[left] > nums[right];};
        priority_queue<int, vector<int>, decltype(cmp)> id(cmp);

        for (int i = 0; i < nums.size(); i++)
        {
            id.push(i);
        }
        for (int i = 0; i < k; i++)
        {
            int updatetarget = id.top();
            id.pop();
            nums[updatetarget] *= multiplier;
            id.push(updatetarget);
        }
        return nums;
    }
};
    • 提交结果
      • 一：584/373个通过的测试样例
    • 分析
      • 优先级队列
      • 错误原因
        • 先pop后push，破坏了排序的稳定性，每次取出的最小元素不是按顺序排列的第一个

    方法三

    class Solution {
public:
    long long quickMul(long long x, long long y, long long m)
    {
        long long res = 1;
        while (y)
        {
            if (y & 1) {
                res = (res * x) % m;
            }
            y >>= 1;
            x = (x * x) % m;
        }
        return res;
    }
    vector<int> getFinalState(vector<int>& nums, int k, int multiplier) {
        if (multiplier == 1)
        {
            return nums;
        }
        long long n = nums.size(), m = 1e9 + 7;
        long long mx = *max_element(nums.begin(), nums.end()); // 取最大值
        vector<pair<long long, int>> v(n);
        for (int i = 0; i < n; i++)
        {
            v[i] = {nums[i], i};
        }
        make_heap(v.begin(), v.end(), greater<>());
        for (; v[0].first < mx && k; k--)
        {
            pop_heap(v.begin(), v.end(), greater<>());
            v.back().first *= multiplier;
            push_heap(v.begin(), v.end(), greater<>());
        }
        sort(v.begin(), v.end());
        for (int i = 0; i < n; i++)
        {
            int t = k / n + (i < k % n);
            nums[v[i].second] = ((v[i].first % m) * quickMul(multiplier, t, m)) % m;
        }
        return nums;
    }
};
    • 提交结果
      • 一：通过
      • 二：通过
    • 分析
      • 通过pair的排序规则来进行堆排序，先比较first，first相同的时候在比较second
      • 对过程进行优化
        • 如果nums中所有数都被更新过了，那么此后更新会从小到大把nums中的所有数都*k
        • 这个过程可直接遍历进行，无需再进行堆排序

      请注意，连线即使在端点也不能相交：每个数字只能属于一条连线。
      以这种方法绘制线条，并返回可以绘制的最大连线数。

    示例

    示例一

    输入：nums1 = [1,4,2], nums2 = [1,2,4]
    输出：2
    解释：可以画出两条不交叉的线，如上图所示。
    但无法画出第三条不相交的直线，因为从 nums1[1]=4 到 nums2[2]=4 的直线将与从 nums1[2]=2 到 nums2[1]=2 的直线相交。

    示例二

    输入：nums1 = [2,5,1,2,5], nums2 = [10,5,2,1,5,2]
    输出：3

    示例三

    输入：nums1 = [1,3,7,1,7,5], nums2 = [1,9,2,5,1]
    输出：2

    提示

    • 1 <= nums1.length, nums2.length <= 500
    • 1 <= nums1[i], nums2[j] <= 2000

    题解

    方法一

    class Solution {
public:

    int maxUncrossedLines(vector<int>& nums1, vector<int>& nums2) {
        int m = nums1.size();
        int n = nums2.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));
        for (int i = 1; i <= m; i++)
        {
            for (int j = 1; j <= n; j++)
            {
                int maxnum = max(dp[i - 1][j], dp[i][j - 1]);
                if (nums1[i - 1] == nums2[j - 1])
                {
                    dp[i][j] = max(maxnum, dp[i - 1][j - 1] + 1);
                }
                else 
                {
                    dp[i][j] = max(maxnum, dp[i - 1][j - 1]);
                }
            }
        }
        return dp[m][n];
    }
};
    • 提交结果
      • 通过
    • 分析
      • 二维数组动态规划
      • 最长公共子序列
      • 转移方程
        • dp[i][j] = max(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) 当num1[i - 1] != num2[j - 1]时
        • dp[i][j] = max(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1] + 1) 当num1[i - 1] == num2[j - 1]时
      • 注意
        • 不要用递归实现
        • nums1与nums2长度不一定相等

  • C++中lambda表达式的用法

    什么是lambda表达式？

    • 一种定义匿名函数对象的简便方法
    • 使用位置
      • 被调用的位置
      • 作为参数传递给函数的位置

    定义

    [capture] (parameters) mutable->return-type {statement}
    • capture
      • [=]通过值捕获所有变量(只有lambda体中提及的对象才会被捕获)
        • [变量名] 通过值捕获指定变量
      • [&]通过引用捕获所有变量
        • [&变量名] 通过引用捕获制定变量
      • this指针
        • 在类成员函数体中使用Lamada表达式，如果需要访问封闭类的成员函数和数据成员，需要将this指针传递给capture子句
        • C++17模式以及更高版本中，可以在capture子句中指定*this通过值捕获this指针，通过值捕获会将整个Lambda闭包复制到调用Lambda的每个站点，在并行或异步操作中非常有用
        [&total, factor]
  [factor, &total]
  [&, factor]
  [=, &total]

        struct S { void f(int i); };

  void S::f(int i) {
      [&, i]{};      // OK
      [&, &i]{};     // ERROR: i preceded by & when & is the default
      [=, this]{};   // ERROR: this when = is the default
      [=, *this]{ }; // OK: captures this by value. See below.
      [i, i]{};      // ERROR: i repeated
  }
    • parameters
      • 类似函数的参数列表
      • C++14中，可传入类型为auto的参数，参数列表中每个auto实例等效于一个不同类型的参数
    • 说明符
      • mutable 允许函数体修改各个复制捕获的对象，以及调用其非const成员函数
      • constexpr 显式指定函数调用运算符为 constexpr 函数。此说明符不存在时，若函数调用运算符恰好满足针对 constexpr 函数的所有要求，则它也会是 constexpr； (C++17 起)
      • consteval 指定函数调用运算符为立即函数。不能同时使用 consteval 和 constexpr。(C++20 起)
    • return-type
      • 前面必须包含’->'

  • C++中priority_queue的用法

    什么是priority_queue?

    • container adaptor
    • 基于堆排序(默认最小堆)
    • 插入和提取的时间复杂度O(log(N))
    • 查找最大/最小元素的时间复杂度O(N)

    构造

    priority_queue <int, vector<int>, greater<int>> gq;
    • int表示在优先级队列中存储的数据类型
    • vector是用于存储这些元素的内部容器类型
    • greater 自定义比较函数，最大堆

    函数

    • empty() 返回队列是否为空
    • size() 返回队列大小
    • top() 返回队列最顶层元素的引用
    • push() 在队列末尾添加元素
    • swap() 交换两个队列的内容
      • 要求类型必须一致，大小可以不一致
    • emplace() 将新元素插入到队列中(就地插入)
    • pop() 用于删除具有最高优先级的元素

    通过lambda表达式自定义比较函数

    auto cmp = [](int left, int right) { return (left ^ 1) < (right ^ 1); };
std::priority_queue<int, std::vector<int>, decltype(cmp)> lambda_priority_queue(cmp);
    • lambda表达式中，如果返回true，说明left的优先级更低(输入是顺序)，返回false，说明right的优先级更低(输入是逆序)
