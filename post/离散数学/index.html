数学 离散数学 笔记

关系

关系及关系性质

  • 二元关系 A*B的子集
  • n元关系 A1,A2…An的子集，Aiis domains, domains的顺序是重要的
  • 补关系 complementary relations
  • 逆关系 inverse relations
  • 函数：特殊的关系
  • 关系A to A
  • 自反：reflexive(irreflexive 反自反)
  • 对称：symmetric(antisymmetry 反对称)(asymmetric 非对称)非对称的关系一定是反自反的
  • 传递：transive Rn包含于R(n=1，2，3…)
  • 关系复合：R2oR1，R1复合R2

关系的表示

  • 0-1矩阵
  • directed graphs

关系闭包closures of relations

  • 自反闭包reflexive closure of R
  • 对称闭包
  • 传递闭包 求Ri的并集(i=1,2..n)；Warshall’s algorithm

等价关系equivalence relations

  • 自反；对称；传递
  • 集合的划分->等价类partitions
  • 商集A/R

偏序关系partial orderings

  • 偏序集合partially ordered sets
  • 偏序关系partial order 自反、反对称、传递
  • 线序关系：可比(comparable)，不可比(incomparable)
  • 全序集合linearly ordered set：所有元素对之间均可比 (linear order全序关系或线序关系->chain)
  • 良序关系well-ordered 所有非空集合都有一个最小元
  • 乘积偏序product partial order
  • 词典顺序lexicographic 先看第一个元素，第一个元素相同再看第二个元素
  • 哈斯图Hasse Diagrams
  • 极大元maximal element
  • 极小元minimal element
  • 最大元greatest element(单位元unit element) 不一定有
  • 最小元least element(零元 zero element) 不一定有
  • 拓扑排序topological sorting（找极小元，找完去掉，继续找极小元）
  • 最小上界LUB
  • 最大下界GLB
  • 格lattices：所有子集都有LUB和GLB
  • 子格sublattice：如果a和b都在原格中，则a和b的LUB和GLB也都在原格中
  • 有界：如果L有最大元和最小元，then lattice L is said to be bounded
  • 如果L是有限格，那么L是有界的
  • 分配格Distributive lattice：最大上界和最小下界的运算符和分配律 -> nondistributive
  • 补元(complement)：如果a和b的最小上界是最大元，最小下界是最小元，则a和b互为补元

半群和群

半群

  • 封闭 closure
  • 二元运算 binary operation
  • 交换律 commutative
  • 结合率 associative
  • 分配律 distributive property
  • 德·摩根定理 De Morgan’s lawns
  • 单位元 Identity for an operation
  • 逆元 Iverse
  • 广群：非空、封闭的
  • 半群：非空、封闭、可结合
  • 单位元、幺元 identity
  • 独异点、含幺半群 monoid
  • 子半群 subsemigroup
  • 同构映射 isomorphism：one to one，f(a * b) = f(a) *’ f(b)
  • 同态映射 homomorphism：everywhere-defined function, f(a * b) = f(a) *’ f(b)
  • products semmigroup 乘积半群
  • quotients of semigroups 商半群
  • 同余关系 congruence relation aRa’ and bRb’ imply (a*b)R(a’*b')
  • 自然同态 natural homomorphism
  • 满同态函数 aRb if and only if f(a) = f(b)
  • 同态基本定理：满同态与商群（自然同态）同构

群

  • 有幺元identity的群成为独异点
  • 半群+幺元
  • 阿贝尔群Abelian：群+交换律
  • 有限群finite group：阶(order)是元素的数目
  • 含有一个元素的群：1种，两个：1种，三个：V和Z4两种
  • S3：f2,f3,f1分别是三角形绕O点旋转120度，240度和360度；g1,g2,g3分别是让三角形关于I1,I2和I3对称
  • 子群subgroup
  • 平凡子群trivial subgroups：本身和只含幺元的集合
  • S3不是阿贝尔群，Z6是阿贝尔群，二者不同构
  • Zm*Zn与Zmn同构当且仅当m与n的最大公因数为1
  • 左陪集left coset aH
  • 右陪集right coset Ha
  • 正规子群normal aH = Ha（集合相等，不一定要每个元素都相等）
  • 阿贝尔群的任意一个子群都是正规子群
  • 核kernel：同态映射中映射到幺元的元素
