    题目

    给你一个字符串 s 、一个字符串 t 。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 "" 。
    注意：

    • 对于 t 中重复字符，我们寻找的子字符串中该字符数量必须不少于 t 中该字符数量。

    示例

    输入：s = “ADOBECODEBANC”, t = “ABC”

    示例 2：

    输入：s = “a”, t = “a”
    输出：“a”
    解释：整个字符串 s 是最小覆盖子串。

    示例 3:

    输入: s = “a”, t = “aa” 输出: "" 解释: t 中两个字符 ‘a’ 均应包含在 s 的子串中，因此没有符合条件的子字符串，返回空字符串。

    leetcode hot100 3 无重复字符的最长子串

    题目

    给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

    示例

    示例一

    输入: s = “abcabcbb”
    输出: 3
    解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

    示例二

    输入: s = “bbbbb”
    输出: 1
    解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。

    示例三

    输入: s = “pwwkew”
    输出: 3
    解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。请注意，你的答案必须是 子串 的长度，“pwke” 是一个子序列，不是子串。

    提示

    • 0 <= s.length <= 5 * 104
    • s 由英文字母、数字、符号和空格组成

    题解

    方法一

    • unordered_map<int, int> pos 记录字符到postion的映射，用于检查字符是否已经出现过
    • 滑动窗口，维护当前的最长子串长度，并在检查到重复的时候判断是否更新maxlen
    class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        if (s.length() < 2)
        {
            return s.length();
        }
        unordered_map<int, int> pos;
        int l = 0, r = 0;
        int maxlen = 0;
        int nowlen = 0;
        for (r = 0; r < s.length(); r++)
        {
            if (pos.find(s[r]) == pos.end())
            {
                pos[s[r]] = r;
                nowlen++;
            }
            else
            {
                maxlen = max(maxlen, nowlen);
                for (; l < pos[s[r]]; l++)
                {
                    nowlen--;
                    pos.erase(s[l]);
                }
                l++;
                pos[s[r]] = r;
            }
        }
        maxlen = max(maxlen, nowlen);
        return maxlen;
    }
};

    leetcode 438 找到字符串中所有字母异位词

    题目

    给定两个字符串 s 和 p，找到 s 中所有 p 的 异位词 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

    示例

    示例一

    输入: s = “cbaebabacd”, p = “abc”
    输出: [0,6]
    解释:

    • 起始索引等于 0 的子串是 “cba”, 它是 “abc” 的异位词。
    • 起始索引等于 6 的子串是 “bac”, 它是 “abc” 的异位词。

    示例二

    输入: s = “abab”, p = “ab”
    输出: [0,1,2]
    解释:

    • 起始索引等于 0 的子串是 “ab”, 它是 “ab” 的异位词。
    • 起始索引等于 1 的子串是 “ba”, 它是 “ab” 的异位词。
    • 起始索引等于 2 的子串是 “ab”, 它是 “ab” 的异位词。

    提示

    • 1 <= s.length, p.length <= 3 * 104
    • s 和 p 仅包含小写字母

    题解

    方法一

    • 分析
      • 用unordered_map<char, int> neednums 记录p中各字母的个数
      • 用unordered_map<char, int> nownums 记录子串中各字母的个数
      • 用重载的运算符==判断两个unordered_map是否相等
      • 注意：second为0的元素要删掉
    class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        unordered_map<char, int> neednums;
        unordered_map<char, int> nownums;
        int modelen = p.length();
        vector<int> ans;
        for (char c: p)
        {
            neednums[c]++;
        }
        int border = s.length() - p.length();
        
        for (int i = 0; i < p.length(); i++)
        {
            nownums[s[i]]++;
        }
        if (nownums == neednums)
        {
            ans.push_back(0);
        }
        for (int i = 1; i <= border; i++)
        {
            nownums[s[i - 1]]--;
            if (nownums[s[i - 1]] == 0)
            {
                nownums.erase(s[i - 1]);
            }
            nownums[s[i + modelen - 1]]++;
            if (nownums[s[i + modelen - 1]] == 0)
            {
                nownums.erase(s[i + modelen - 1]);
            }
            if (nownums == neednums)
            {
                ans.push_back(i);
            }
        }
        return ans;
    }
};

    方法二

    • 分析
      • 用vector differnum(26);记录当前子串与p的字母数量的差异
      • 用int differ记录有差异的字母的个数(注意 不是记录总的字母数量之差)
      • 每次移动滑动窗口的时候更新differnum，并更新differ
      • differ为0的时候，该子串即为符合要求的子串
      • 重点是各字符的数量差异而不是数量本身
    class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        vector<int> differnum(26, 0);
        int Slen = s.length();
        int Plen = p.length();
        vector<int> ans;
        if (Slen < Plen)
        {
            return ans;
        }
        for (int i = 0; i < Plen; i++)
        {
            differnum[s[i] - 'a']++;
            differnum[p[i] - 'a']--;
        }
        int differ = 0;
        for (int i = 0; i < 26; i++)
        {
            if (differnum[i])
                differ++;
        }
        if (!differ)
        {
            ans.push_back(0);
        }
        int len = Slen - Plen;
        for (int i = 1; i <= len; i++)
        {
            differnum[s[i - 1] - 'a']--;
            if (!differnum[s[i - 1] - 'a'])
            {
                differ--;
            }
            else if (differnum[s[i - 1] - 'a'] == -1)
            {
                differ++;
            }
            differnum[s[i + Plen - 1] - 'a']++;
            if (!differnum[s[i + Plen - 1] - 'a'])
            {
                differ--;
            }
            else if (differnum[s[i + Plen - 1] - 'a'] == 1)
            {
                differ++;
            }
            if (!differ)
                ans.push_back(i);
        }
        return ans;
    }
};

  • leetcode hot100 49 字母异位词分组

    题目

    给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。
    字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

    示例

    示例 1:

    输入: strs = [“eat”, “tea”, “tan”, “ate”, “nat”, “bat”]
    输出: [[“bat”],[“nat”,“tan”],[“ate”,“eat”,“tea”]]

    示例 2:

    输入: strs = [""]
    输出: [[""]]

    示例 3:

    输入: strs = [“a”]
    输出: [[“a”]]

    提示

    • 1 <= strs.length <= 104
    • 0 <= strs[i].length <= 100
    • strs[i] 仅包含小写字母

    题解

    方法一

    • 分析
      • 用一个int记录一个字符串中所有出现的字母
      • 将int映射到这一类字符串在ans中的位置
    • 问题
      • 使用的字母相同不能保证两个字符串相同
    • 提交结果
      • 106/126个通过的测试样例
      • 解答错误
    class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        vector<vector<string>> ans;
        unordered_map<int, int> pos; // 从字符串的值映射到在ans中的位置
        for (string str: strs)
        {
            int nowbits = 0;
            for (char c: str) // 计算值
            {
                nowbits = nowbits | (1 << (c - 'a'));
            }
            if (pos.find(nowbits) != pos.end())
            {
                ans[pos[nowbits]].push_back(str); // 将字符串插入已经有的数组中
            }
            else
            {
                ans.push_back({str});
                pos[nowbits] = ans.size() - 1;
            }
        }
        return ans;
    }
};

    方法二

    • 分析
      • 将所有字符串转换为统一格式，映射到其在ans数组中的位置
    • 提交结果
      • 通过
    class Solution {

        unordered_map<string, int> pos; // 从字符串的值映射到在ans中的位置
        for (string str: strs)
        {
            string newstr = str;
            sort(newstr.begin(), newstr.end());
