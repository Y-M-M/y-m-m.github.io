  • mac下终端常用命令

    关于目录的操作

    • ls
      • ls: 查看当前目录下文件
      • ls-a: 查看当前目录下文件以及隐藏文件
    • mkdir: 创建新目录
    • cd
      • cd 目录名: 进入目录
      • cd ..: 返回上一级目录
    • rm
      • rmdir 目录名: 删除空目录，此删除不会出现在废纸篓里
      • rm -rf 目录名: 删除空目录/非空目录，此删除不会出现在废纸篓里(强制删除，不会询问)
      • rm -r 目录名: 删除目录，会提示你确认删除文件
      • rm 文件名: 此删除不会出现在废纸篓里
    • cp
      • cp 文件名1 文件名2: 拷贝文件1，并命名为文件2
      • cp 文件名 文件夹名/: 将文件拷贝到文件夹中
      • cp -r 文件夹名1 文件夹名2: 将文件夹1拷贝为文件夹2
    • mv
      • mv 文件夹名 文件夹名/: 将一个文件夹移动到另一个文件夹中
    • find *.txt: 查找当前目录下所有.txt文件
    • pwd: 显示当前目录
    • open
      • open .: 打开成可视化的文件夹(当前目录)
      • open 路径: 打开路径对应的文件夹

    关于文件的操作

    • touch 文件名: 创建文件
    • mv 文件名1 文件名2: 将文件重命名为文件名2
    • less 文件路径
      • 查询文件内容
      • 按q退出
    • cat 文件名: 显示文件内容

    关于系统的操作

    • 关机
      • sudo shutdown -h now立刻关机
      • sudo shutdown -h +10十分钟后关机
      • sudo shutdown -h 20:00晚上八点关机
    • sudo shutdown -r now重启
